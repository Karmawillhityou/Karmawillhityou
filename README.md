## 🛡️ Data Center Security & Infrastructure

Interesado en cómo se diseñan, administran y protegen los **centros de datos**, el núcleo de la infraestructura digital moderna.

### 🔐 Protección de infraestructuras críticas

* Hardening de servidores
* Segmentación de redes
* Control de accesos

### 🌐 Gestión de redes en centros de datos

* Arquitecturas de red
* VLANs y segmentación
* Alta disponibilidad

### 🛡️ Defensa y monitorización

* Detección de intrusiones
* Análisis de tráfico
* Monitorización de logs

### 💾 Gestión y protección de datos

* Backups y redundancia
* Recuperación ante desastres
* Integridad de evidencias digitales

### 🔎 Investigación y análisis forense

Uso de herramientas como **Autopsy** para investigar sistemas comprometidos, analizar imágenes de disco y reconstruir incidentes de seguridad.

### ⚙️ Laboratorios personales

Creación de entornos virtualizados para experimentar con:

* Redes seguras
* Análisis forense
* Simulación de incidentes
* Defensa de infraestructuras
